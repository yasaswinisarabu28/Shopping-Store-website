A responsive web application built using HTML, CSS, and JavaScript that fetches product data from the DummyJSON API and dynamically displays products in an attractive card-based layout. The application demonstrates API integration, asynchronous JavaScript, DOM manipulation, template cloning, and responsive UI design.

Features
Fetches product data from the DummyJSON API.
Dynamically generates product cards using JavaScript.
Displays product image, title, category, price, rating, and description.
Responsive design for desktop, tablet, and mobile devices.
Clean and modern user interface.
Error handling for API requests.
Uses reusable HTML templates for efficient rendering.
Technologies Used
HTML5
CSS3
JavaScript (ES6+)
Fetch API
DummyJSON REST API
Learning Outcomes
Working with REST APIs.
Handling asynchronous operations using Promises and Fetch.
Manipulating the DOM dynamically.
Creating reusable UI components.
Building responsive web interfaces.
